# krypnova
Krypnova is the first autonomous financial brain for crypto, stocks, and commodities. Powered by Exion, our AI engine with reinforcement learning and deep neural networks, it simulates, reasons, and executes strategies 24/7 across major exchanges, managing risk and portfolios with hedge fund precision.

# Contributing to Krypnova

- Open issues for bugs, doubts or suggestions.
- Fork and open a pull request for any code contribution.
- Please describe your changes and test before submitting.
- Be respectful and collaborative with the community.
